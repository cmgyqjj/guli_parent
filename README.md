# guli后端部分

## 简介
guli在线教育项目是尚硅谷开发的一套在线教育系统，使用SpringBoot+SpringCLoud实现。
涵盖了SpringBoot,MyBatis,Redis,Mysql,MyBatis-plus，swagger等技术。
